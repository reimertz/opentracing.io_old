
OpenTracing.io


### Prerequisits

you need to have node.js, npm, jekyll installed on your computer.

### Install dependencies
```
npm install
```

### run development
```
npm run dev
open http://localhost:3000 #
```

##Gitbook

### run gitbook
```
npm run docs:watch
```

### build documentation for static site (http://opentracing.io)
```
npm run docs:build
```
